How to use these

To brief a creator: send them the hub link (or a single share link). It opens directly, shows every frame, the on-frame text to write, and links to the exact Masonry B-roll clips in Drive.
To change a caption or timecode: the simplest way is to say the change and it gets pushed for you. Or use the Edit link (opens the source on GitHub, needs a GitHub account with write access to the repo). Live pages update about a minute after any edit, on the same URLs.
Note on the blue text boxes: on a share page they are for preview only, typing there does not save for other viewers. Final wording is set in the source and republished, so everyone always sees the current version.
